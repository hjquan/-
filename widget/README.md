### 项目结构

```
├── commonpage               # 
├── commonUi                 # h5的ui组件
├── css                      # 不参与编译的资源文件
├── html                     # 企业端页面文件夹
├── image                    # 图片资源
├── res                      # 
├── script                   # 公共函数
│   ├── common.js            # 封装部分公共函数，以及请求地址
├── wgt                      # 项目端页面文件夹
```

### apiCloud

1.拉去代码 apiCloud开发工具（代码管理/代码检出/APICloud云端应用）
2.apiCloud开发工具下，根目录鼠标右键gi提交代码，提交成功后（代码管理/同步到云端）
    注：vscode不能提交代码。
